# 🛒 EcommerceHub — Enterprise Multi-Vendor E-commerce Platform

EcommerceHub is a **production-grade full-stack multi-vendor marketplace** built using **Spring Boot**, **React**, and **Redux**.  
It supports separate **Customer**, **Seller**, and **Admin** portals with real-time inventory management, secure authentication, and Stripe payment integration.

---

## 🚀 Features

### 🏪 Full-Stack Marketplace
- Multi-vendor e-commerce platform with modular architecture
- Separate portals for:
  - 👤 Customer
  - 🧑‍💼 Seller
  - 🛡️ Admin
- Real-time product and inventory management

### 🔐 Advanced Security
- Implemented enterprise authentication using **Spring Security**
- **JWT Token based authentication**
- **Role-Based Access Control (RBAC)** for authorization

### 💳 Payment Integration
- Integrated **Stripe Payment Gateway**
- Secure checkout flow and payment verification

### 🗃️ Scalable Database Design
- Designed complex entity relationships using **JPA/Hibernate**
- Database: **PostgreSQL**
- Implemented:
  - Pagination
  - Sorting
  - Search / Filtering
